# 디지털 문방구 
 
</br>

> ## 프로젝트 소개 
```

태블릿용 템플릿 무료 공유 웹 사이트

```

</br>

> ## 프로젝트 목적
```

태블릿 사용자 수가 늘어나면서 다양한 목적의 템플릿 수요가 급증하고 있다.
하지만 한 번에 많은 종류의 무료 템플릿을 볼 수 있는 사이트는 소수이다.
따라서 학습, 일정, 업무 등 자신의 필요에 맞는 템플릿을 쉽고 다양하게 찾을 수 있는 웹 사이트를 개발했다. 

```

</br>
